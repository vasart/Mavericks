Mavericks
=========

Arduino-based remote controlled bot.
Android application is used to send commands.

<img src="https://raw.github.com/vasart/Mavericks/master/bot.jpg" width="600">

Components:
* Arduino Uno
* WiFi shield
* 2 x servo drives
* Sonar
* IR sensor
* 2 x 9v batteries
* Breadboard
* Custom power distribution board
* Some random case
